# DermAI by Skana

A web-based AI dermatology assistant that helps users identify potential skin conditions through image analysis. DermAI combines two custom-trained deep learning models with a conversational AI assistant to provide structured, accessible skin health insights.

> **Medical Disclaimer:** DermAI is a research and educational tool. Results produced by the AI models are not a substitute for professional medical diagnosis. Always consult a qualified dermatologist or medical professional for any skin health concerns.

---

## What It Does

DermAI provides three core AI-powered services:

- **Skin Cancer Detection** вЂ” Upload a dermoscopic image and receive a binary classification (benign or malignant) with a confidence score, powered by a custom-trained ResNet-50 model.
- **Skin Disease Classification** вЂ” Upload a skin lesion image and receive a multi-class prediction across 8 ISIC 2019 disease categories, with per-class confidence bars and a clinical severity score, powered by a custom-trained ResNet-152 model.
- **Karl вЂ” AI Dermatology Chatbot** вЂ” A conversational assistant powered by Gemma (via Google AI Studio) that answers dermatology-related questions. Strictly scoped to skin health topics with built-in security guardrails.
- **AI Disease Descriptions** вЂ” After a disease classification, users can request an AI-generated plain-language explanation of their result, generated by Gemini (via Google AI Studio).

## Quick Start

### Prerequisites

- Python 3.12 (conda recommended)
- Node.js 18+ and npm
- Google AI Studio API key(s) вЂ” [aistudio.google.com](https://aistudio.google.com)

### 1. Backend Setup

```bash
# Navigate to the backend folder
cd backend

# Install Python dependencies
pip install -r requirements.txt

# Copy the environment variable template and fill in your API keys
copy .env.example .env
# Open .env and add your Google AI Studio keys

# Start the API server
uvicorn main:app --reload --port 8000
```

### 2. Frontend Setup

```bash
# Navigate to the frontend folder
cd frontend

# Install Node dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at **http://localhost:3000**.

## Model Files

The trained model weight files are not included in this repository due to file size. Place the model files in `backend/models/`:

| File | Purpose |
|---|---|
| `best_model.pth` | Cancer detection model (ResNet-50) |
| `best_model.pt` | Disease classification model (ResNet-152) |

---

## API Keys

Three Google AI Studio API keys are required. Add them to `backend/.env`:

```
GEMINI_PRIMARY_API_KEY=your_key_here
GEMINI_BACKUP_API_KEY=your_backup_key_here
GEMMA_API_KEY=your_key_here
```

See `backend/.env.example` for the full template.

## Dataset

The disease classification model was trained on the **ISIC 2019** skin lesion dataset, covering 8 dermoscopic disease categories. The cancer detection model was trained on a binary (benign/malignant) dermoscopic image dataset.
